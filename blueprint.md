# Blueprint: Biorhythm Lotto Number Recommender

## 1. Overview
A simple web application that recommends lotto numbers based on the user's biorhythm, calculated from their birth date.

## 2. Features
*   **Biorhythm Calculation**: Calculates physical, emotional, and intellectual cycles.
*   **Lotto Recommendation**: Generates 5 sets of lotto numbers (6 main + 1 bonus) using a seed derived from the biorhythm.
*   **Responsive Design**: Modern and clean UI that works on both desktop and mobile.
*   **SEO Optimized**: Includes meta tags and JSON-LD for better search engine visibility.

## 3. Technical Stack
*   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla).
*   **Deployment**: Can be hosted on any static web hosting service (e.g., Firebase Hosting, GitHub Pages).

## 4. Implementation Details
*   `index.html`: Structure, Styling, and Logic (Self-contained for this implementation).
